## Overview
This project aims to automate various tasks in Gmail, making your email experience more efficient and hassle-free. Whether it's managing your inbox, sending scheduled emails, or organizing your messages, this automation tool has got you covered!

## Features
- **Automatic Email Sorting**: Keep your inbox organized by automatically sorting incoming emails into folders.
- **Scheduled Emails**: Send emails at a specified time without having to remember to do it manually.
- **Bulk Email Sender**: Easily send personalized bulk emails without the hassle of copy-pasting.
- **Custom Notifications**: Get notifications for important emails based on specific criteria.

## Getting Started
To get started with the Gmail Automation Project, clone the repository and install the required dependencies:

```bash
git clone https://github.com/Sudharsan68/gmail_automation.git
cd gmail_automation
pip install -r requirements.txt
```

## Usage
Simply run the main script to start automating your Gmail tasks:

```bash
python main.py
```
